# Clipy

A lightweight, native macOS clipboard manager that lives in a floating drawer at the edge of your screen. Built entirely in Swift and SwiftUI — no Electron, no web views, no background daemons.

**Requires macOS 14 (Sonoma) or later · Apple Silicon + Intel (universal binary)**

---

## Features

### Clipboard History
- Monitors the system clipboard continuously and saves every copied item
- Supports **text**, **links**, **images**, **documents**, and files of any size
- Each card shows the **source app icon**, **app name**, and the exact **date and time** it was copied
- Re-paste any item with a single click; or **drag it directly** from the drawer into any app
- Right-click any card for a context menu: Paste, Pin / Unpin, Remove

### Smart Categorisation
- Items are automatically sorted into: **Recent**, **Pinned**, **Text**, **Links**, **Images**, **Documents**
- Category filter bar lets you jump to any section instantly
- **Search** across all items — text content, URLs, and file names for images and documents

### Pinning
- Pin any item to keep it permanently, safe from automatic pruning
- Pinned items have their own configurable count limit (or unlimited)
- Pinned items are excluded from Clear History unless explicitly included

### Large File Support — Zero Memory Overhead
- Files copied from Finder store only the **file path reference** — the file bytes are never read into memory
- A **512 px preview thumbnail** is generated asynchronously: images are downscaled directly, videos use AVFoundation (frame 0), PDFs and other types use QuickLook, unknown types fall back to the system file icon
- Drag and drop works for **files of any size** — the original file path is handed to the destination app
- Inline image data (screenshots, browser images) is stored in a local BlobStore (`~/Library/Application Support/Clipy/Blobs/`) — blobs are cleaned up automatically when items are pruned or deleted

### Drawer UI
- Full-width frosted-glass **NSPanel** that appears at the top or bottom edge of your screen
- Collapses into a floating **pill widget** when you switch to another app
- The pill widget can be **dragged anywhere** on screen; its position is remembered between sessions
- Cards feature a **dark header partition** showing the source app icon, name, and timestamp — separated from the content preview by a white divider line
- Smooth spring animations, hover scale effects, and a 3D card-flip on tap
- Custom scroll indicator with position tracking

### Settings
- **Launch at Login** — start Clipy automatically on boot via `SMAppService`
- **Retention period** — keep items for 1–365 days; older items pruned on every new copy
- **History limit** — keep the last N items (1–999) or unlimited
- **Pinned limit** — cap pinned items (1–999) or unlimited
- **Drawer position** — top or bottom, with a frosted glass visual picker
- **Keyboard shortcut** — fully customisable recorder (default: **⌥⇧V**)
- **Clear History** with optional "also clear pinned items" checkbox
- **Reset to Defaults** restores all settings in one action

### Privacy
- Everything stays on your Mac — no network requests, no telemetry, no cloud sync
- Clipy writes a marker type (`org.nspasteboard.Clipy`) so other clipboard managers skip its own writes
- `org.nspasteboard.ConcealedType` (password managers), `TransientType`, and `AutoGeneratedType` are silently ignored
- Dynamic (`dyn.*`) and Microsoft OLE source types are filtered out

### Notifications
- Optional system notification when a new item is saved (standard macOS permission prompt)

## Building from Source

```bash
git clone https://github.com/hewhocodes/clipy.git
cd clipy
swift build -c release --arch arm64 --arch x86_64
```

To run in development:

```bash
swift run
```

### Producing a DMG

The CI workflow (`.github/workflows/build.yml`) assembles a `.app` bundle and wraps it in a DMG automatically on every version tag push (`v*`).

To do it locally:

```bash
swift build -c release --arch arm64 --arch x86_64

APP="dist/Clipy.app"
mkdir -p "$APP/Contents/MacOS" "$APP/Contents/Resources"
cp .build/apple/Products/Release/Clipy "$APP/Contents/MacOS/Clipy"
cp Resources/Clipy.entitlements "$APP/Contents/Resources/"

codesign --force --deep --sign - \
  --entitlements Resources/Clipy.entitlements \
  --options runtime \
  "$APP"

hdiutil create -volname Clipy -srcfolder "$APP" -ov -format UDZO dist/Clipy.dmg
```

The app uses **ad-hoc code signing** (`--sign -`) so no Apple Developer account is required.

> **Gatekeeper note:** macOS will block an ad-hoc signed app downloaded from the internet. To open it: right-click → Open, or run `xattr -d com.apple.quarantine Clipy.app` in Terminal.

## Project Structure

```
Sources/Clipy/
├── ClipyApp.swift                 App entry point, Window definitions
├── Notifications.swift            All NotificationCenter name constants
├── Models/
│   ├── ClipboardItem.swift        SwiftData model for a single clipboard entry
│   └── ClipboardCategory.swift    Category enum (text, link, image, doc, other)
└── Services/
│   ├── PasteboardMonitor.swift    Polls NSPasteboard, captures and categorises items
│   ├── PasteboardWriter.swift     Writes a history item back to NSPasteboard
│   ├── HistoryStore.swift         SwiftData CRUD and pruning
│   ├── BlobStore.swift            On-disk storage for large binary items
│   ├── Categorizer.swift          Classifies pasteboard items by content type
│   ├── ImagePreviewLoader.swift   Loads and resolves image data from any item
│   ├── ThumbnailGenerator.swift   Async thumbnail generation (image/video/QL/icon)
│   ├── AppIconCache.swift         Caches source app icons by bundle ID
│   ├── DrawerPresenter.swift      NSPanel lifecycle, expand/minimise, positioning
│   ├── DrawerWindow.swift         AppKit panel, hosting view, widget overlay
│   ├── HotkeyManager.swift        Global keyboard shortcut via KeyboardShortcuts
│   ├── LaunchAtLoginService.swift SMAppService launch-at-login wrapper
│   ├── ClipyNotifier.swift        macOS notification support
│   └── Settings.swift             UserDefaults-backed observable settings model
└── Views/
    ├── ClipboardDrawerView.swift  Drawer: header, search, category filter, card grid
    ├── RecentItemCard.swift       Individual clipboard card with source app header
    ├── ClipboardDragSupport.swift AppKit drag source and pasteboard writer
    ├── DrawerRootView.swift       Root view wired to presenter notifications
    ├── MinimizedDrawerStrip.swift Pill widget shown when drawer is collapsed
    ├── ClipyGlass.swift           Frosted-glass confirmation dialog overlay
    └── SettingsView.swift         Settings form
```
